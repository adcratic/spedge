Bulk SMS Gateway service
Bulk SMS Gateway service is classified into two types, Transactional sms service and Promotional bulk sms services.

Transactional SMS Service tend to be informational in nature. Transactional sms gateway is used for various alert sms messages like welcome sms on user's registration, Verifying mobile number using OTP SMS from user, transaction alert sms from bank, train and bus reservation alert, sending notifications and important information to users / employees using SMS Gateway user interface or Transactional SMS api code. Transactional sms API code helps developers to achieve automation of SMS communications to users.

Promotional sms gateway is used mostly for marketing and advertisement purpose. Promotional sms is become most popular because it is quickest way to promote any product or service. FAQs ABOUT BULK SMS SERVICES ↴

Bulk Messaging is the delivery of SMS messages to multiple mobile numbers at the same time. Bulk sms can be automated by integration of sms APIs in your software such as CRM / ERP or website based on some event. Ex. An automated confirmation sms can be sent on new customer registration on your website/CRM.

End-user mobile receives SMS with a header/sender name which looks like TM-SPEDGE or TM-060410. That is called as Header ID. You can have header name similar to your company or brand name (6 characters) like TD-SPEDGE. Header must be approved on any one operator's DLT platform to configure for your SMS account. You may require to complete your company KYC to get registered on MNO's DLT platform. Multiple headers can be added with a single SMS account.

Maximum text sms length can be up to 2500 chars. Initial 160 characters of sms is equivalent to 1 sms credit, If it exceeds 160 characters then the message consumes 2 or more sms credits. The character length for 2 messages is 306, for 3 messages is 459, for 4 messages is 612, and for 5 messages is 724 and so on..

Yes, Transactional / informative messages can be delivered on DND numbers 24/7. There is no restrictions for event based transactional and informative services messages to get delivered on NDNC registered numbers. However there are some restrictions for Advertisement / promotinal messages as per TRAI regulations.

You can receive sms online using Virtual Numbers. Virtual number can be a 10 digit mobile number which is called Long code or a short code number. A callback url or API can be configured in virtual number account to automatically forward all the incoming messages to your server/platform.

Yes it is possible to send and deliver non-english SMS text to mobile numbers. It is called Unicode messaging. You need to choose unicode option while sending sms or send extra flag as unicode true with sms API. Every 70 characters is charged as 1 sms credit in unicode messaging.
Difference Between Promotional SMS and Transactional Messaging
Transactional SMS is informative messages ex. booking and order alerts.
Messages are delivered to all numbers both DND and non-DND numbers.
Transactional SMS are delivered 24x7 with no time restriction.
Transactional SMS can use a 6-alpha Sender ID for your business. Ex. TM-
SPEDGE
Registration on MNO's DLT platform is mandotory with Header and content templates.
Delivery reports and analytics are available for transactional bulk sms.
Promotional SMS is marketing messages like offers, discounts or promotions.
Messages are only delivered to non-DND numbers.
Promotional SMS can be sent only between 9:00 AM and 9:00 PM.
Promotional bulk SMS will be delivered with a random 6-digit Sender ID Ex. TM- 573627
Registration on MNO's DLT platform is mandotory with Header and consent templates.
Delivery reports are not availble for promotional bulk SMS, As per TRAI regulations.
